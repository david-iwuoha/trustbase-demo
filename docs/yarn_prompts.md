Response Determination Logic
The demo adapter uses keyword matching to determine which response to return:

Banking queries: Keywords like “bank”, “first bank”, “financial”, “loan”, “transaction” → banking_data response
Telecom queries: Keywords like “mtn”, “telecom”, “phone”, “network”, “usage” → telecom_data response
Rights queries: Keywords like “rights”, “consent”, “withdraw”, “delete”, “revoke” → consent_rights response
Access explanation: Keywords like “explain”, “why”, “access”, “purpose” → explain_access response
Default: All other queries → default response
